# API test


## ต้องการ
สำหรับการสร้างและใช้งานแอปพลิเคชันที่คุณต้องการ:
- JDK21
- maven 3.9.9

## การติดตั้ง

1.โคลนโปรเจคต์นี้ไปยังเครื่องของคุณ:

```bash
git clone https://github.com/aekkaphonz/backend.spring.git
```


## การรันแอปพลิเคชัน
1.เริ่มต้นเซิร์ฟเวอร์ด้วยคำสั่ง:
```bash
mvn spring-boot:run
```
2.เปิดเบราว์เซอร์และไปที่:
`http://localhost:8080`
